# Aerial-Object-Classification-And-Detection
This project focuses on developing a deep learning system capable of distinguishing between birds and drones in aerial images. With the increasing use of drones in surveillance, security, and commercial applications, it has become important to automatically identify flying objects in the sky. Birds and drones can sometimes appear visually similar in aerial imagery, which makes manual identification difficult. This project aims to address that challenge using computer vision and deep learning techniques.
Models Used:
Convolutional Neural Network(Cnn)
Transfer Learning Model(MobileNetV2)
